HOW TO RUN THE UNIVERSITY ERP APP

1. Database Setup:
   - Open your MySQL Workbench/Client.
   - Ensure user 'root' has whatever password the user has in MySQL (update src/main/java/edu/univ/erp/util/DatabaseSetup.java).
   - Run the 'DatabaseSetup' class (edu.univ.erp.util.DatabaseSetup) to initialize tables and seed users.

2. Running the App:
   - Run 'Main' class (edu.univ.erp.Main).
   - The Login Screen will appear.
