University ERP Project
======================

Prerequisites:
- Java 21
- Maven
- MySQL Server (running on port 3306)

------------------------------------------------------------------------
STEP 1: CONFIGURATION
------------------------------------------------------------------------
1. Open the file: src/main/resources/application.properties
2. Change 'db.user' and 'db.password' to match your local MySQL credentials.
   (Make sure your MySQL server is running first!)

------------------------------------------------------------------------
STEP 2: DATABASE SETUP (Run once)
------------------------------------------------------------------------
Open a terminal in this folder and run these commands:

  mvn clean compile
  mvn exec:java -Dexec.mainClass="edu.univ.erp.util.DatabaseSetup"

Wait for the message: "âœ… Database Setup Complete!"

------------------------------------------------------------------------
STEP 3: RUNNING THE APP
------------------------------------------------------------------------
Start the application with:

  mvn exec:java -Dexec.mainClass="edu.univ.erp.Main"
